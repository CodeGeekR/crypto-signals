## 🚀 Quick Start

### Prerequisites:

- Node.js 18+
- npm 8+

### Installation:

```bash
# Clone the repository
git clone https://github.com/CodeGeekR/crypto-signals.git

# Install dependencies
cd crypto-signals
npm install

# Start development server
npm run dev
```
